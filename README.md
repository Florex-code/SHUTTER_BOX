# SHUTTER_BOX (Dark theme) — Fresh Starter

## Quick start

1. Copy `.env.example` to `.env` and add your Unsplash token:
   `VITE_UNSPLASH_TOKEN="your_access_token_here"`

2. Install deps:
   ```bash
   npm install
   ```

3. Start dev server:
   ```bash
   npm run dev
   ```

The app uses Vite + React + Tailwind + Framer Motion. Likes are stored in localStorage under `shutter_favs`.
